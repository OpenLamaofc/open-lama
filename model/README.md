# 🧠 AI Models Directory

Place your `.gguf` model files in this directory to use them with **OPEN LAMA**.

## 📥 Recommended Model
For the best balance between performance and quality on local machines, we recommend:

**Llama 3.2 3B Instruct (Q5_K_S)**
- **Download Link**: [HuggingFace - bartowski/Llama-3.2-3B-Instruct-GGUF](https://huggingface.co/bartowski/Llama-3.2-3B-Instruct-GGUF/blob/main/Llama-3.2-3B-Instruct-Q5_K_S.gguf)
- **Official Website**: [openlama.fun](https://openlama.fun)

## 🛠️ Instructions
1. Download the `.gguf` file from the link above.
2. Move the file into this `model/` folder.
3. Restart **OPEN LAMA** and select the model from the boot menu.

> [!NOTE]
> You can add multiple models to this folder. OPEN LAMA will allow you to choose which one to load upon startup.
